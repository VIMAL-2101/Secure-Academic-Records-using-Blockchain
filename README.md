# 🔐 Secure Academic Records using Blockchain

A full-stack application that ensures **secure, tamper-proof academic record management** using Blockchain technology.

---

## 🚀 Features

* 🔒 Blockchain-based record storage
* 👨‍🎓 Student academic data management
* 👩‍🏫 Teacher verification system
* 🛡 Immutable audit logs
* ⚡ FastAPI backend with REST APIs
* 🌐 Modern frontend interface

---

## 🧱 Tech Stack

### 💻 Backend

* Python (FastAPI)

* Blockchain (Smart Contracts)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/VIMAL-2101/Secure-Academic-Records-using-Blockchain.git
cd Secure-Academic-Records-using-Blockchain
```

---

### 2️⃣ Backend Setup

```
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

---

### 3️⃣ Frontend Setup

```
cd frontend
npm install
npm run dev
```
